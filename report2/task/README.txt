作業分別為不同資料夾裡面的Practice Quiz.py和Graded Quiz.py，而More.py是Graded Quiz.py的另一種版本。

使用方法:
1.打開CMD
2.>>> cd "要測試的py檔所在的絕對路徑"
3.>>> python "py檔的檔名" < "同目錄下的txt檔的檔名"
(雙引號不能省略，否則 "Practice Quiz.py" 這樣的檔案名稱會被切成 "Practice" 這個Token，而找不到檔案。)

另外的使用方法:
1.打開CMD
2.>>> python "py檔的絕對路徑" < "同目錄下的txt檔的的絕對路徑"
(雙引號同樣不能省略)
